# TradersMate

Kleines eigenstaendiges Web-Tool fuer Star-Citizen-Handelsrouten, Komponenten, Armor, Waffen und Verbrauchsgütern.

## Starten

Oeffne `index.html` direkt im Browser.

## Bedienung

1. Startstation auswaehlen.
2. Material auswaehlen.
3. TradersMate zeigt alle passenden Verkaufsstellen mit Gewinn oder Verlust pro SCU und Marge.

Die Daten liegen lokal in `data.js` und wurden aus der UEX Corp API 2.0 erzeugt. Enthalten sind sichtbare Waren, Commodity-/Admin-Terminals und die zugehoerigen Commodity-Preiszeilen. Preise in Star Citizen aendern sich schnell; pruefe groessere Ladungen vor dem Vollkauf kurz im Terminal oder in UEX.

Das Tool läuft lokal auf dem PC, daten sich einmal am Tag beim öffnen mit UEX ab um aktuell zu sein
